# BMH Reborn

## Vision

BMH Reborn ist eine moderne Open-Source-Neuimplementierung des Klassikers
„Bundesliga Manager Hattrick“.

Das Projekt soll das ursprüngliche Spielgefühl erhalten, gleichzeitig aber
eine moderne Architektur besitzen und langfristig erweiterbar sein.

Es handelt sich ausdrücklich nicht um einen 1:1-Klon, sondern um eine
Neuimplementierung mit ähnlichem Gameplay.

---

## Projektziele

- klassische Managermechanik
- schnelle Saisons
- historisch spielbare Datenbanken
- Mod-Unterstützung
- vollständige Savegames
- Multiplayer als langfristiges Ziel

---

## Technischer Stack

- .NET 9
- C#
- Godot 4 (Frontend)
- xUnit
- GitHub Actions
- JSON-Datenbanken
