# Contributing

## Branches

main bleibt jederzeit lauffähig.

## Commits

Konvention:

feat:
fix:
refactor:
test:
docs:
chore:

## Tests

Neue Features sollten durch Unit-Tests begleitet werden.

## Code Style

- Nullable aktiviert
- keine öffentlichen Felder
- IReadOnlyCollection statt List
- keine statischen Services
- XML-Kommentare für öffentliche APIs
